1. adding: INSERT INTO
   updating: UPDATE
   delete: DELETE FROM

2. INSERT INTO table_name (column_x, column_y) VALUES (data, data);

   After INSERT INTO the table name is written, followed by a list of the
   columns of data to be inserted (enclosed in parenthesis). If the
   column names are omitted SQL assumes the data is being listed in the
   order of the columns as they exist in the table. Lastly VALUES is
   followed by a parenthesis enclosed/comma separated list of the data
   to be added corresponding to the columns. One row of data can be added
   or multiple rows.

   UPDATE table_name SET column = data WHERE id=1234;

   UPDATE begins the statement followed by the table name. SET comes next
   and one or multiple columns can be listed to pass in new data for
   the column(s). WHERE is used to match the record or records to be
   changed.

   DELETE FROM table_name WHERE column = data

   DELETE FROM is followed by the table name. The WHERE clause can match
   a single row or many rows, any matching row(s) will be deleted.

4. - Data types
   first and last name : text
   rsvp                : boolean
   guests              : integer
   meals               : numeric(3,2)

   - Create table

   CREATE TABLE wedding (
        first_name text,
        last_name  text,
        rsvp       boolean,
        meals      numeric(3,2)
   );


   - Add column for Thank Yous

   ALTER TABLE wedding ADD COLUMN thank_yous boolean;

   - Remove meals column

   ALTER TABLE wedding DROP COLUMN meals;

   - Add table number columns

   ALTER TABLE wedding ADD COLUMN table_number integer;

   - Delete wedding table

   DROP TABLE wedding;
